# CHANGELOG Generator Skill

Automatically generate a structured `CHANGELOG.md` from your git history using Claude.

## Quick Start (3 Steps)

1. **Install**: Copy `generate_changelog.py` and `changelog.sh` to your project root.
2. **Auth**: Ensure `ANTHROPIC_API_KEY` is in your environment.
3. **Run**: Execute `bash changelog.sh`.

## Requirements
- Git
- Python 3
- Anthropic API Key (optional, fallbacks to raw list)
